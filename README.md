# patrones-de-diseno
Este repositiorio es una investigación sobre los patrones de diseño en este caso basandome en la documentación brindada por el profesor  https://refactoring.guru/es/design-patterns, tendremos código en lenguaje Python, también  tendremos diagramas de los códigos y un documento con descripción de cada patrón de diseño
## Investigación sobre Patrones de Diseño

Este repositorio es una investigación sobre los **patrones de diseño** en el desarrollo de software, basada en la documentación brindada por el profesor y la referencia principal de [Refactoring Guru - Patrones de Diseño](https://refactoring.guru/es/design-patterns).

### Contenido del repositorio

- **Implementaciones en Python:** Ejemplos prácticos de cada patrón de diseño utilizando el lenguaje Python.
- **Diagramas:** Diagramas explicativos para visualizar la estructura y funcionamiento de cada patrón.
- **Documento descriptivo:** Un documento con la descripción y análisis de cada patrón de diseño.

### Objetivo

El objetivo de este repositorio es comprender, analizar y ejemplificar los principales patrones de diseño, facilitando su estudio y aplicación en proyectos reales.

### Categorías de patrones

#### 🏗️ Patrones Creacionales
Proporcionan mecanismos de creación de objetos que incrementan la flexibilidad y reutilización del código.

#### 🔧 Patrones Estructurales  
Explican cómo ensamblar objetos y clases en estructuras más grandes, manteniendo flexibilidad y eficiencia.

#### 🎯 Patrones de Comportamiento
Tratan con algoritmos y la asignación de responsabilidades entre objetos.
